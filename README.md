# Video_To_MIDI
Video To MIDI - Open source project 

MIDI channels 
Brightness - note
Red - 18
Green - 19
Blue - 20
Radius - 21

Loop Midi used to interface between programe and DAW - This is a discontined exturnal project - If a alternative is known please suggest. 
https://www.tobias-erichsen.de/software/loopmidi.html
